This directory contains line data pertaining to COVID-19 in the state of Florida. The data is separated by day, and is added to daily from the following source:

https://open-fdoh.hub.arcgis.com/datasets/florida-covid19-case-line-data

As of 07/06/2020, the amount of data is too big to be stored in the repository. A way to get around this will be looked into.
